# Daily Email Summarizer — AI Concierge Agent Using ADK

This repository contains the code and notebook for the Agents Intensive Capstone project demo: a Daily Email Summarizer that fetches unread emails (CSV fallback on Kaggle), summarizes them, assigns priorities, and extracts action items.

## Files
- `email_summarizer.ipynb` — runnable Kaggle notebook (demo)
- `data/sample_emails.csv` — sample data for reproducibility
- `agent/` — agent code and helpers
- `diagrams/` — generated images for media gallery

## How to run
Open `email_summarizer.ipynb` in Kaggle or Colab and run all cells. The notebook will generate `daily_report.json` and `daily_report.md`.
